AntaleVPN IP API

Raw IP addresses list for API integration

API Endpoint

Use the following endpoint to retrieve a raw list of IP addresses:

GET https://list.antalevpn.com/?=api-list

Returns a plain text response with IP addresses separated by newlines.

Example Response

The API endpoint returns a clean list of IP addresses without any HTML formatting:

188.146.24.33
95.111.206.247
200.234.224.147

Usage Examples

# Using curl
curl https://list.antalevpn.com/?=api-list

# Using JavaScript fetch
fetch('https://list.antalevpn.com/?=api-list')
  .then(response => response.text())
  .then(data => console.log(data));

# Using Python requests
import requests
response = requests.get('https://list.antalevpn.com/?=api-list')
print(response.text)
